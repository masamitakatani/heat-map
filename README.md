# heat-map

AIエンジニアで自動生成されたプロジェクトです。

## プロジェクト情報

- **作成日**: 2025/11/1
- **生成ツール**: AIエンジニア VSCode拡張機能

## フォルダ構成

```
heat-map/
├── phases/          # Phase実行結果
├── backend/         # バックエンドコード
├── frontend/        # フロントエンドコード
├── docs/            # ドキュメント
└── .ai-engineer.json # プロジェクト設定
```

## 開発手順

1. VSCodeで「AIエンジニア: Phase実行」コマンドを実行
2. Phase 1から順番に実行していく
3. 各Phaseの成果物は `phases/phase-X/` に保存される
4. 自動生成されたコードを確認・テストする

## Phase一覧

- Phase 1: 要件定義
- Phase 2: 技術選定
- Phase 3: データベース設計
- Phase 4: API設計
- Phase 5: フロントエンド設計
- Phase 6: 実装（バックエンド）
- Phase 7: 実装（フロントエンド）
- Phase 8: テスト
- Phase 9: デプロイ設定
- Phase 10: 本番デプロイ
- Phase 11: ドキュメント作成
